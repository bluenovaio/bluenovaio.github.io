## Installing
Follow the steps below to install Otto locally and begin otto-generating tests!

:::tip
Make sure that you have already run npm init or have a node_modules folder or package.json file
in the root of your project to ensure otto is installed in the correct directory.
:::

### npm install
Install Otto via npm:

```jsx
  cd /your project/path
```

```jsx
  npm install otto
```

### yarn add
Install Otto via yarn:

```jsx
  cd /your project/path
```

```jsx
  yarn add otto
```
